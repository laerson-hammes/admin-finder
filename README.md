# Admin Finder

You can use **-hn** or **--host** to specify the hostname and, **-w** or **--wordlist** to specify the wordlist file name or path

> **Example:** python .\admin-finder.py -hn google.com -w subdomains.txt

For help, use **-h** or **--help**

> **Example:** python .\admin-finder.py -h

Observation:

If you do not specify the hostname or the wordlist, you will be prompted
